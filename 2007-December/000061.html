Log:
changes in flags
 - resized Angola and Japan flags to be same size as others
 - fixed colors in Findland flag (it was shown as a black square)
 - cleanup of the svn properties of the flag images
 - added Spain flag (found in http://www.clickfire.com/content/graphics/favicons/flags_icons.php)
 - added Bulgaria flag (thanks to Adrian Dimitrov)

Note: The idea is to not add more flags to the game this way, but to allow the players to create their own flag and be downloaded automatically by other players.
